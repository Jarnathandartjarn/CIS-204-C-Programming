# CIS-204-C-Programming

## Instructions to submit an assignment
### 1. `git add . `
```
This adds all of your changes to the current/new commit
```
### 2. `git commit -m "Your commit message"`
```
Creates the commit and tags it with a descriptive message
```
### 3. `git push origin main`
```
This is the step that actually pushes or publishes your changes to github (origin) and the branch (main) that you specify
```
### 4. `git status`
```
Shows which file you are tracking and the state of your current change.
```
